A current example of use: Watermarking

Steganography is not used by apprentice spies wanting to hide their secret information. It is also used in today's industry.
Its main application is watermaking. This consists in hiding a copyright in a protected work.
Thus, in case of copyright dispute, the watermark will be to show the originality of the work.

Want to try the LSB Method ?

Cloaked-Pixelis a public git repository and a platform independent Python tool to implement LSB image steganography and a basic detection technique.
Features:

  • Encrypt data before insertion.
  • Embed within LSBs.
  • Extract hidden data.
  • Basic analysis of images to detect LSB steganography.
Here is a link of the git repository https://github.com/livz/cloacked-pixel
